A simple **Expense Tracker Web Application** built using **Spring Boot**, **Thymeleaf**, and **MySQL**.  
This project helps users **add, view, and delete expenses** in an easy and organized way.

 Project Overview

The **Expense Tracker** allows users to:
- Add daily expenses
- View all recorded expenses
- Delete expenses when not needed
- Manage expenses using a clean and colorful UI

While i'm building this project i got small practice on Spring MVC, Crud operations, @RestAPI's...
Features

  Add new expenses (amount, category, description, date)
  View all expenses in a table format
  Delete expenses using ID
  User-friendly and responsive UI
  Data stored securely in MySQL database

Tech Stack Used

| Backend | Spring Boot (Java) |
| Frontend | HTML, CSS, Bootstrap, Thymeleaf |
| Database | MySQL |
| ORM | Spring Data JPA |
| Build Tool | Maven |
| Server | Embedded Tomcat |
